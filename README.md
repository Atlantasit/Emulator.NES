## Compatibility
For a list of games known to be playable, visit [the wiki page](https://github.com/Xyene/Emulator.NES/wiki/Games-Known-to-Work).

Currently, the following mappers are implemented:

* 0 - NROM
* 1 - MMC1
* 2 - UxROM
* 10 - MMC4
* 94 - Senjou no Ookami
* 155 - MMC1A
* 180 - Crazy Climber
